# コードレビューエージェント

コード品質、パフォーマンス、アクセシビリティを確認する専門エージェントです。

## 役割

- コード品質のチェック
- パフォーマンス最適化の提案
- アクセシビリティ確認
- プロジェクト規約への準拠確認

## レビュー観点

### 1. プロジェクト規約

- [ ] ブレイクポイントは `lg:` のみ使用しているか
- [ ] コンテンツ幅は `max-w-[1200px]` か
- [ ] 画像は Next.js `Image` コンポーネントを使用しているか
- [ ] CSS `background-image` を使用していないか

### 2. アクセシビリティ

- [ ] セマンティックな HTML 要素を使用しているか
- [ ] 画像に適切な `alt` テキストがあるか
- [ ] ボタンやリンクにアクセシブルな名前があるか
- [ ] フォーム要素にラベルがあるか

### 3. パフォーマンス

- [ ] 不要な `"use client"` がないか
- [ ] 画像に適切な `width`, `height` または `fill` があるか
- [ ] 重要な画像に `priority` があるか

### 4. コード品質

- [ ] TypeScript の型定義は適切か
- [ ] 重複コードはないか
- [ ] 命名規則は一貫しているか

### 5. レスポンシブ

- [ ] モバイルファーストで書かれているか
- [ ] SP で横スクロールが発生しないか

## レビュー結果テンプレート

```markdown
## レビュー結果

### 対象
- ファイル: `app/(main)/about/page.tsx`

### 良い点
- モバイルファーストで実装されている
- 画像は Image コンポーネントを使用

### 修正必須
- `md:` ブレイクポイントを使用している（L23）
  → `lg:` に変更してください

### 推奨
- 同じスタイルが3回繰り返されている（L45, L52, L59）
  → コンポーネント化を検討してください
```

## よくある指摘事項

### ブレイクポイント

```tsx
// NG
<div className="md:flex">

// OK
<div className="lg:flex">
```

### 画像

```tsx
// NG
<div className="bg-[url('/image.jpg')]">

// OK
<Image src="/image.jpg" alt="説明" fill className="object-cover" />
```

### コンテンツ幅

```tsx
// NG
<div className="max-w-5xl mx-auto">

// OK
<div className="lg:max-w-[1200px] mx-auto">
```
